About Data

This dataset contains data of the passengers in titanic ship. Here, we have several columns which hold some information about passenger
such as name, age, sex, fare, class and many more which we mention below with brief explanation.

Dataset Columns

  • PassengerId : Passenger unique ID
  • Survived : Survival, where 0 means not survived and 1 means survived.
  • Pclass : Ticket class, where 1 means 1st(Upper) class, 2 means 2nd(Middle) class, and 3 means 3rd(Lower) class,
  • Name : Passenger's Name
  • Sex : Passenger's gender
  • Age : Passenger's Age in years
  • SibSp : Number of siblings / spouses aboard the Titanic
  • Parch : Number of of parents / children aboard the Titanic
  • Ticket : Ticket number
  • Fare : Passenger fare
  • Cabin : Cabin number
  • Embarked : Port of Embarkation, where C = Cherbourg, Q = Queenstown, S = Southampton
